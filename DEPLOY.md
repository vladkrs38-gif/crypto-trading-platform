# Деплой на сервер — простыми словами

## Что это

**Деплой** = отправить твой код на сервер и обновить сайт proplatforma.ru.

---

## Как деплоить

### Вариант 1: Сказать AI-агенту (рекомендуется)

Напиши в чате:

> **«Деплой проект на сервер»**  
> или  
> **«Задеплой проект»**

AI-агент запустит скрипт и всё сделает сам.

---

### Вариант 2: Вручную

Открой PowerShell в папке проекта и выполни:

```powershell
cd c:\plat3_1\plat3
.\deploy.ps1
```

Скрипт:
1. Скопирует код (app, src, конфиги) на сервер
2. Соберёт проект на сервере
3. Перезапустит сайт

---

## Когда деплоить

- **Не нужно** после каждого мелкого изменения — разрабатывай локально (`npm run dev`)
- **Нужно**, когда фича готова и хочешь обновить сайт на proplatforma.ru

---

## Технические детали (для AI-агента)

- Сервер: root@89.191.226.213
- Путь: /var/www/plat3
- Команда: `.\deploy.ps1` или вручную: scp app, src + npm run build + pm2 restart plat3-next
- Не трогать .env.production на сервере
