# Быстрый старт

## 1. Установка зависимостей

```bash
npm install
```

## 2. Запуск проекта

```bash
npm run dev
```

## 3. Откройте в браузере

http://localhost:3000

## Что дальше?

1. **Выберите пару** - нажмите на селектор в хедере и выберите одну из топ-10 пар
2. **Выберите таймфрейм** - используйте селектор таймфреймов для выбора интервала
3. **200-тиковый график** - выберите "200 тик" в селекторе таймфреймов для просмотра тикового графика

## Особенности

- ✅ Автоматическое подключение к Binance WebSocket
- ✅ Обновление данных в реальном времени
- ✅ 200-тиковый график (сбор тиков локально)
- ✅ Стандартные таймфреймы (1m, 5m, 15m, 1h, 4h, 1d и др.)
- ✅ Темный интерфейс в стиле GitHub

## Структура

- `src/components/` - React компоненты
- `src/lib/binance.ts` - Binance API коннектор
- `src/store/useTradingStore.ts` - Управление состоянием (Zustand)
- `src/styles/globals.css` - Стили и цветовая схема
