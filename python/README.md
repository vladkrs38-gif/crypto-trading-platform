# Density Tracker

Трекер плотностей для мониторинга крупных лимитных ордеров на Binance и Bybit.

## Особенности

- **Отслеживание времени жизни** — показываются только плотности, которые стоят минимум 1 час
- **Подсчёт касаний** — записывает когда цена подходит к плотности
- **Персистентность** — данные сохраняются в SQLite, переживают перезагрузку
- **REST API** — для интеграции с фронтендом

## Запуск

### Windows
```
start_tracker.bat
```

### Вручную
```bash
pip install -r requirements.txt
python api_server.py
```

## API Endpoints

### GET /api/densities
Получить активные плотности.

**Параметры:**
- `min_lifetime` — минимальное время жизни в секундах (default: 3600)
- `symbol` — фильтр по символу (например, BTCUSDT)
- `exchange` — фильтр по бирже (binance/bybit)
- `density_type` — фильтр по типу (buy/sell)
- `min_amount` — минимальная сумма в USD
- `max_distance` — максимальная дистанция в %
- `limit` — максимальное количество

### GET /api/densities/{id}
Получить детальную информацию о плотности с историей касаний.

### GET /api/stats
Статистика трекера.

### GET /api/coins
Список отслеживаемых монет.

## Конфигурация

Редактируйте `config.py`:

```python
MIN_LIFETIME_SECONDS = 3600  # 1 час минимум
MIN_DENSITY_USD = 100_000    # $100K минимум
MIN_VOLUME_24H = 100_000_000 # $100M объём монеты
MAX_DISTANCE_PERCENT = 5.0   # До 5% от цены
```

## База данных

Файл `densities.db` создаётся автоматически и содержит:
- Все обнаруженные плотности
- Историю касаний
- Время появления и исчезновения

## Автозапуск (Windows)

1. Win+R → `shell:startup`
2. Создать ярлык на `start_tracker.bat`

Или через Task Scheduler для запуска при старте системы.
